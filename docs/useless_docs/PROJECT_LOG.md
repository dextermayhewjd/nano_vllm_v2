## TX: （standard format）

**构建了什么：**
（客观事实，做了什么）

**为什么这样做：**
（关键设计决策和理由）

**学到了什么：**
（核心概念，3-5 条）

**容易踩的坑：**
（常见错误，意外的地方）

**过程中的发现：**
（实现过程中遇到的意外，标注类型：）
（A = 阻塞了当前 ticket，当场修了）
（B = 以后会需要，记到 TICKETS.md 的 Backlog）
（C = 之前 ticket 的 bug，建了补丁 ticket）

**我的理解：**
（你自己用中文写 2-5 句。不能空。）

**盲区：**
（每 3 个 ticket 做 quiz 后才填。平时留空。）
